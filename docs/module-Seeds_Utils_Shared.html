Determines what kind of seed is going to need the current model. There are two kind of seeds:

  • The simple, which only runs a function that create fake data by using the seed JSON file for the given seed model.
  • The complex, which also create fake models for the given seed model, however, this one also takes in consideration every related models by using the relations of the current model.

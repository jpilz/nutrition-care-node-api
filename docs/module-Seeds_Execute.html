Seeds/Execute

Creates fake data based on the seed JSON files that were created by the prepare-seeds.js script. When a user runs this script, the user has to enter the wanted Model to seed and the number of wanted records. This file uses the faker.js library to generate all the dummy data.

Get all the Seed models and processes them to only get the model name, the model filename and the model properties to push it inside the arrayModels array.

Example

arrayModels: [
   {
    "filename": "seed-administrator.json",
    "name": "Administrator",
    "properties_seeds": [
      {
        "name": "{{random.words(3)}}"
      }
      //More properties...
     ]
   }
  //More seed models...

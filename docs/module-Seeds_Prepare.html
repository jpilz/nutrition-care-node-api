Seeds/Prepare

Prepares all the JSON seed files in order to generate fake data for the Loopback's model that are allocated on the common folder. The output that generate is a JSON file per Loopback model file that allows the user to enter faker.js' mustache syntax to generate all kind of fake data to test his or her applications.

    "name": "Model1",
    "properties_seeds": [
